## 1. Configuration
*1. Create environment*
```
# Create environments
python -m venv venv
source venv/bin/activate  # or .\venv\Scripts\activate in Window
```

*2. Install requirements*
```
python -m pip install -U pip setuptools
pip install -r requirements.txt
```
